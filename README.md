# ColorCopy

画面のカラーをピックしてクリップボードにコピーします。

## 使用方法

1. ColorCopy.exe を起動します。
2. コピーするテキストの形式を選択します。
3. ダイアログ上で左クリックダウンし、ドラッグを開始します。
4. ピックしたいカラーのところで左クリックアップし、ドラッグを終了します。
5. ピックしたカラーが選択した形式でクリップボードにコピーされます。
